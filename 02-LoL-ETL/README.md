# League of Legends ETL Pipeline

## Overview
This project implements an end-to-end ETL pipeline using the Riot Games API to
extract, transform and load League of Legends match data for analytical purposes.

The dataset focuses on Ranked Solo/Duo, Ranked Flex and Normal Draft matches
from January 2025 onwards.

## Data Source
- Riot Games API (Match-V5)

## Pipeline Structure
- **Extract**: Fetch match data for multiple summoners and queues
- **Transform**: Parse raw JSON into a structured tabular format
- **Load**: Generate an analysis-ready CSV dataset

## Tech Stack

- Pandas
- Riot API
